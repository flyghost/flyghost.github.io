# 1、临时（只在该终端生效）

- 第一种方法：走http

```shell
export http_proxy=http://127.0.0.1:1080
export https_proxy=http://127.0.0.1:1080
```
- 第二种方法：走socks
```shell
export http_proxy=socks://127.0.0.1:1080
export https_proxy=socks://127.0.0.1:1080
```

- windows终端则将export改为set
